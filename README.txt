In order to configure this bot replace the BOTTOKEN variable in main.py with your own discord bot token.
https://discordpy.readthedocs.io/en/stable/discord.html
Custom made discord music bot to use with for any server (NON COMMERICAL)!!

Commands Supported:
-play Play songs and add to queue, input is keywords or links
-pause Pause current song
-resume Resume current song
-skip Skip current song
-queue View queued songs
-remove Remove songs from queue, input is index of queue
-shuffle Shuffle queue
-dc Disconnect Bot
-stop Another way to Disconnect Bot
-purge purges messages
